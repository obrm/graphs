# Graphs View

A React with TypeScript app that shows a Line Graph using 'recharts' package and a Gantt Graph using 'react-google-charts' package.

## Website
https://obrm-graphs.netlify.app

## Usage

### Install Dependencies 

```
npm install
```

### Run (:3000)

```
npm run start
```

## Build & Deploy

# Create prod build

```
npm run build
```
